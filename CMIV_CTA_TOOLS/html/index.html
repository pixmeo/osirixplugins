Description: This plug-in was designed on a core function called “virtual contrast injection” algorithm [1] [2] [3], which aims to separate different contrast filled structures or other high intensity structures from each other using the connectivity of the contrast in the vessel lumen. Unlike most conventional segmentation algorithm, this algorithm doesn’t focus on defining the exact edge of the vessels, which means there will be some soft tissue surrounding the vessels in the segment results. The task of classifying different components of different intensity is left to some conventional visualization techniques such as maximum intensity projection (MIP) or volume rendering (VRT). The centerlines of all visible branches will be extracted automatically after the segmentation with a new algorithm also based on the connectedness theory. Some general visualization techniques, such as multiplanar reformatting (MPR), curved plane reformatting (CPR) and VRT, are also included in this plug-in. 3D paths for CPR and masks(brush ROI) for segmental VRT can also be defined manually which allow users to access those general visualization function without using our segmentation and skeletonization algorithm.

Main Features: arbitrary angle 2D Orthogonal MPR - arbitrary angle 2D curved MPR and straightened CPR - Interactive vessel segmentation - Automatic centerline extraction - Segmental Volume Rendering
